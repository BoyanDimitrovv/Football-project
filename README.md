# Football-project
⚽ Football Chatbot – Stage 2

Описание:

Това е Python чатбот, свързан с SQLite база данни.
Позволява управление на футболни клубове чрез текстови команди.

Функционалности:
- Добавяне на клуб
- Показване на списък с клубове
- Изтриване на клуб
- Обновяване на клуб (по желание)
- Помощна команда
- Изход
- Логване на командите в commands.log

Как се стартира:

1. Изпълнете sql/schema.sql чрез DB Browser for SQLite или SQLite CLI.
2. Отворете терминал и стартирайте чатбота:
   cd src
   python main.py

Примерни команди:

Добави клуб Левски
Добави клуб Ботев
Покажи всички клубове
Изтрий клуб Ботев
помощ
изход

Използвани технологии:
- Python 3
- SQLite
- Regex за разпознаване на команди

Структура на проекта:

src/        → Python код
sql/        → Database schema
docs/       → Документация и ER диаграми
commands.log → Лог на командите
README.md   → Инструкции и описание
.gitignore  → Изключени файлове
