# Football-project
Този проект е базов Python чатбот, свързан с SQLite база данни.
Позволява управление на футболни клубове чрез текстови команди.

Функционалности:
CRUD операции за таблица clubs (добавяне, показване, изтриване, обновяване)
Чатбот с разпознаване на команди чрез regex
Логване на командите в commands.log
Команди за помощ и изход
Как се стартира:

1️⃣ Създаване на базата данни:
Изпълнете файла sql/schema.sql чрез DB Browser for SQLite или SQLite CLI.

2️⃣ Стартиране на чатбота:
Отворете терминал и изпълнете:

cd src
python main.py

След това чатботът ще стартира и ще приема команди.

Примерни команди:
Добави клуб Левски
Добави клуб Ботев
Покажи всички клубове
Изтрий клуб Ботев
помощ
изход


Използвани технологии:
Python 3
SQLite
Regex за разпознаване на команди

Структура на проекта:
src/        → Python код (main.py, chatbot.py, clubs_service.py, db.py)
sql/        → Database schema (schema.sql)
docs/       → Документация и ER диаграми
commands.log → Лог на командите
README.md   → Инструкции и описание
.gitignore  → Изключени файлове

Минимална функционалност (Pass condition):

зход от програмата
