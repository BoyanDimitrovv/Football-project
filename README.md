# Football-project
Проектът „FootballAI Assistant“ представлява интелигентна футболна информационна система, разработена на Python, която използва SQL база данни и включва AI модул за анализ и прогнози на мачове.
	Системата позволява управление на футболни данни чрез чатбот интерфейс, който разпознава текстови команди на естествен език и извършва съответните операции върху базата данни.

 2.Цел на проекта
-Да се разработи релационна база данни за управление на футболна информация.
-Да се реализират CRUD операции чрез чатбот.
-Да се изгради AI модул за прогнозиране на резултати.
-Да се приложат знания от Python, SQL и изкуствен интелект.

3. Основни функционалности
	Системата включва следните модули:
Управление на клубове, Управление на футболисти, История на трансфери, Управление на лиги, Мачове и събития, Класиране, Статистика и AI прогнози.
4. Архитектура на системата
Системата е изградена модулно:
-DB слой – връзка с база данни
-Business логика – CRUD операции
-AI слой – анализ и прогнози
-Chatbot слой – обработка на текстови команди

Как се стартира:
1. Изпълнете sql/schema.sql чрез DB Browser for SQLite или SQLite CLI.
2. Отворете терминал и стартирайте чатбота:
   cd src
   python main.py

Примерни команди:
Добави клуб Берое
Добави клуб Левски
Покажи всички клубове
Изтрий клуб Ботев
помощ
изход

Използвани технологии:
- Python 3
- SQLite
- Regex за разпознаване на команди
